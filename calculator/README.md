# 🧮 CLI Calculator

A professional and interactive Command Line Interface (CLI) calculator built with TypeScript and Node.js. Perform basic arithmetic operations with style!

## 🚀 Features
- **Addition**: Sum two numbers.
- **Subtraction**: Find the difference between two numbers.
- **Multiplication**: Multiply two numbers.
- **Division**: Divide one number by another.
- **Modulo**: Find the remainder of a division.
- **Interactive Interface**: Colorful prompts and animations.
- **Banner Display**: Beautiful ASCII banner at startup.

## 🛠️ Technologies
- **TypeScript**: For type-safe development.
- **Node.js**: The runtime environment.
- **Inquirer**: For interactive user prompts.
- **Chalk**: For styling and coloring the terminal output.
- **Node-Banner**: To display a professional startup banner.
- **Chalk-Animation**: For smooth terminal animations.

## 📖 How to Run
Run the project directly from the root directory using Node.js:

```bash
node calculator/index.ts
```

Or using `tsx`:

```bash
pnpm dlx tsx calculator/index.ts
```
