# 🎲 CLI Number Guessing Game

A fun and interactive CLI-based number guessing game where you try to match a hidden number.

## 🚀 Features
- **Random Number Generation**: The system picks a number between 1 and 6.
- **Interactive Prompts**: Rainbow-colored input field for a fun experience.
- **Instant Feedback**: Immediate results on whether you guessed right or wrong.
- **Professional Banner**: Welcomes you with a stylish ASCII art banner.

## 🛠️ Technologies
- **TypeScript**: Ensuring robust code with type safety.
- **Node.js**: Powering the backend logic.
- **Inquirer**: Managing user input through the terminal.
- **Chalk**: Enhancing visual feedback with vibrant colors.
- **Node-Banner**: Providing a polished startup appearance.
- **Chalk-Animation**: Adding dynamic rainbow animations.

## 📖 How to Run
Run the project directly from the root directory using Node.js:

```bash
node number-guessing-game/index.ts
```

Or using `tsx`:

```bash
pnpm dlx tsx number-guessing-game/index.ts
```
